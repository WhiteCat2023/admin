## Overview

The Ariba Admin Dashboard is a comprehensive reporting and management system designed to streamline administrative operations, monitor activities, and manage user access. Built with modern web technologies, it provides real-time insights and control over core business functions.

## Key Features

### 📍 Location Tagging
- Tag and manage locations across your network
- Geospatial organization of resources and activities
- Location-based reporting and analytics

### 📊 Reports List
- Comprehensive report generation and management
- Customizable report viewing and filtering
- Export capabilities for data analysis
- Historical report tracking

### 📈 Analytics
- Real-time dashboard analytics and metrics
- Performance tracking and visualization
- Trend analysis and data insights
- Customizable analytics widgets

### 🆘 SOS Requests
- Emergency request management system
- Real-time SOS alert tracking
- Priority-based request handling
- Response time monitoring

### 👥 User Management
- Complete user administration interface
- Role-based access control
- User activity monitoring
- Permission management

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone <repository-url>
   cd ariba/admin
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the development server
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint to check code quality
